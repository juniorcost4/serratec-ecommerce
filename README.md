## Instalação

Para executar o Projeto siga esses passos.

1. Clone o repositório

 ```sh
 git clone https://github.com/juniorcost4/serratec-ecommerce.git
 ```
2. Instale os pacotes do projeto
  
  * npm
    ```sh
    npm install
    ```

  * yarn
    ```sh
    yarn
    ```
    
3. Execute o projeto
  
  * npm
    ```sh
    npm run dev
    ```

  * yarn
    ```sh
    yarn dev
    ```
